<project_specification>
  <project_name>WebDevCody.com</project_name>
  
  <overview>
    A personal website for YouTuber WebDevCody (Cody) to showcase his web development content, sell courses, promote products/services, and engage with his community. The site serves as a central hub connecting visitors to his YouTube channel, educational courses, side projects, and provides tools for community engagement including newsletter subscription and video suggestion submissions.
  </overview>
  
  <technology_stack>
    - **Framework**: Next.js 14 (App Router)
    - **Language**: TypeScript 5
    - **Styling**: TailwindCSS 3.3 with tailwindcss-animate
    - **UI Components**: Radix UI (Dialog, Dropdown Menu, Label, Slot, Toast)
    - **Animation**: Framer Motion 12
    - **Icons**: Lucide React
    - **Forms**: React Hook Form with Zod validation (@hookform/resolvers)
    - **State Management**: TanStack React Query 5
    - **Content**: next-mdx-remote for MDX support
    - **Theming**: next-themes for dark/light mode
    - **Environment Variables**: @t3-oss/env-nextjs with Zod validation
    - **Security**: next-recaptcha-v3 for spam protection
    - **Cloud Services**: AWS SDK (DynamoDB for data storage)
    - **Image Processing**: Sharp
    - **Notifications**: Sonner for toast notifications
    - **Mobile UI**: Vaul for drawer components
    - **Typography**: @tailwindcss/typography plugin
    - **Fonts**: Inter (Google Fonts)
  </technology_stack>
  
  <core_capabilities>
    1. **Landing Page**: Hero section with embedded YouTube video, newsletter signup, video suggestion CTA, and about/skills section
    2. **Newsletter Subscription**: Email collection with reCAPTCHA protection, integrated with external mailing list service
    3. **Courses Showcase**: Display paid courses available on Gumroad with descriptions and purchase links
    4. **Products Portfolio**: Showcase of side projects (Project Planner AI, Icon Generator AI)
    5. **Contact Hub**: Links to Discord, YouTube, GitHub, and email contact
    6. **Video Suggestions**: Feedback form for community video topic requests (integrates with Project Planner AI)
    7. **Dark/Light Mode**: Full theme support with system preference detection
    8. **Responsive Design**: Mobile-first with dedicated mobile menu and drawer components
    9. **Legal Pages**: Terms of Service, Privacy Policy, Fulfillment Policy
    10. **Analytics**: Event tracking via SendEventOnLoad component
  </core_capabilities>

    **Environment Variables**
    - TABLE_NAME: DynamoDB table name
    - RECAPTCHA_SECRET: Server-side reCAPTCHA key
    - MAILING_LIST_ENDPOINT: External mailing list API
    - MAILING_LIST_PASSWORD: API authentication
    - NEXT_PUBLIC_RECAPTCHA_SITE_KEY: Client-side reCAPTCHA key
    - NEXT_PUBLIC_PLANNER_ID: Project Planner AI integration ID
  </data_models>
  
  <api_design>
    **Server Actions**
    
    1. `subscribeAction({ token, email })`
       - Purpose: Subscribe user to newsletter
       - Validation: reCAPTCHA verification
       - Integration: External mailing list API (POST with Bearer auth)
    
    **External API Integrations**
    
    1. Mailing List Service
       - Endpoint: `MAILING_LIST_ENDPOINT`
       - Method: POST
       - Auth: Bearer token
       - Body: `email` (form-urlencoded)
    
    2. Project Planner AI Feedback
       - Endpoint: `https://projectplannerai.com/api/feedback`
       - Method: POST
       - Body: `{ name, feedback, projectId }`
    
    3. Google reCAPTCHA v3
       - Client: next-recaptcha-v3 provider
       - Server: Token verification via `verifyRecaptcha()`
    
    **External Links (Not APIs)**
    - YouTube: youtube.com/@webdevcody
    - GitHub: github.com/webdevcody
    - Discord: discord.gg/N2uEyp7Rfu
    - Gumroad: webdevcody.gumroad.com
    - Video Suggestions: suggestions.webdevcody.com
  </api_design>
